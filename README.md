# 🌍 Flask Web Server  

## 📌 Overview  
This is a simple **Flask-based web server** with basic routes. It serves a greeting message at different endpoints.  

## 🚀 Features  
- ✅ **Minimal Flask Application**  
- 🔄 **Multiple Routes (`/` and `/bye`)**  
- ⚡ **Lightweight & Fast**  

## 🛠️ Technologies Used  
- **Python** 🐍  
- **Flask** 🚀
